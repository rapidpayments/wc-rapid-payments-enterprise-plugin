Take instant eft or credit card payments on your WooCommerce store using Callpay.

== Description ==
This plugin allows for Credit/Cheque Card Payments, Instant EFT Payments as well as Alternative Payment Methods (APMâ€™s).

Accept ABSA, Standard Bank, Capitec, Investec, FNB, Nedbank, Bidvest Bank, Tyme Bank, African Bank and Old Mutual Bank payments directly into your bank account as well as the credit card payment.

== Installation ==
Goto Plugins > WC Rapid Payments Enterprise Plugin > Settings
Insert API Username and Password
Set optional parameters
Save settings.

Gateway should appear on checkout if configured with correct credentials.
